Projeto construido durante o primeiro ano da faculdade de Análise e Dev. de Sistemas utilizando como principais linguagens o HTML, CSS, Bootstrap, JAVA e SQL.

### Etapas do projeto:
- Construção de um site simples de tela inicial, dashboard, onde o cliente tem a visualização das principais informações sobre sua conta;
- Construção de um Banco de Dados relacional, onde todos os dados do cliente necessários para abertua de conta e utilização de serviços ficam armazenados;
- Desenvolvimento do modelo relacional do Banco de Dados no Oracle SQL DataDeveloper;
- Construção da parte física do banco de dados no Oracle DataBase utilizando o SQL;
- Integração do Banco de Dados com o JAVA, utilizando frameworks para a conexão e consceitos de Persistence.
